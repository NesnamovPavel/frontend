Lesson 1: HTML intro

HTML: HyperNext Markup Language - язык разметки гипертекста

Узнать об элементе ссылке в MDN
  • h1 - heading, заголовок первого порядка, должен быть на странице один
  • src атрибут для адреса источника изображения
  • alt - текст- фдьтернатива кортинки, который увидит пользователь, если картинка на подгрузится
  • p - текстовый элемент параграф, отличается размером м стилизацией
  • target="_blank" - открывает ссылку в новой вкладке
  • ul неупорядоченный список, состоит из li- элементов списка lists
