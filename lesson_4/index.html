Значения по позиционированию:

  • flex - отображение для родителя позиционируемых элементов
  • gap- с помощью этого свойства в flex можно указывать промежуток между элементами (альтернатива для margin)
  • spacy-evenly- свойство при котором распределяет равномерно пространство вокруг элементов
  • space-between- равномерно распределяет пространство между элементами
  • start / end- перемещает в начало или в конец контейнера
  • justiffy-content- свойство выравнивания по главной оси
  • align-items- свойство позиционирования по поперечной оси
  • column- меняет оси позиционирования местами, а в месте с осями меняются и свойства позиционирования
  • flex-dorection-свойство, меняющее оси местами
  • gap-промежуток между элементами в flex-контейнере
  • align-content-управдяет группой элементов при переносе через flex-wrap
